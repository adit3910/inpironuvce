Event Name

Merge It

Team size: 2

Ticket Price: Rs.100 (per Team)

Cash Prize: Upto 5k

Mode: Offline

Round 1: Blind Coding

Participants are given a problem statement, where participants will write the code without looking at the monitor.

Time duration: 30 minutes

Round 2: Optimazation

Shortlisted teams should optimize the code written in the Round 1 by eliminating errors in a short period of time.

Time duration: 30 minutes

Round 3: Switch coding

A team member will be given a problem statement, he / she will code for the problem for 15 mins. After 15mins, the other team member will swap places and continue to code. Switching will takes place after every 15mins. Code will be judged based on the number of errors, quality, time taken to code and output.

Time duration: 1 hour
